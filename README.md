# Stock News Sentiment Analyzer

Live: [https://your-app.streamlit.app](https://your-app.streamlit.app)

## Features
- Multiple tickers (input or CSV)
- Finviz, Google News, Yahoo Finance
- VADER + FinBERT sentiment
- Async + cached = **fast**
- Progress bar + status

## Deploy

1. Push to GitHub
2. Go to [streamlit.io/cloud](https://streamlit.io/cloud)
3. New app → connect repo → `app.py`
4. Done!

> First run: `playwright install chromium`
